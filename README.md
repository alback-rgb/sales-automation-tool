# 📊 Sales Automation Tool

A practical automation tool that processes sales reports and sends a clear summary via email.

Built to help small businesses save time and better understand their sales data.

---

## 🚀 Features

- Reads sales data from Excel files  
- Cleans and processes raw data  
- Aggregates total sales per product  
- Generates a clear and readable summary  
- Sends the report via email  

---

## 🛠️ Tech Stack

- Python  
- Pandas  
- smtplib  
- python-dotenv
